Инструкция по запуску:

Скомпилировать программу с помощью javac: javac MergeSortFiles.java

Запустить программу с указанными параметрами. Примеры запуска:

java MergeSortFiles -i -a output.txt input1.txt input2.txt
Для сортировки целых чисел по возрастанию. Результат будет записан в output.txt находящиеся в input1.txt и input2.txt.

java MergeSortFiles -s -d sorted_output.txt file1.txt file2.txt file3.txt
Пример сортировки строк по убыванию. 
Результат будет записан в sorted_output.txt из файлов file1.txt, file2.txt и file3.txt.
